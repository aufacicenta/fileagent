production.

## Running Project

### Prerequesites

- [Node.js](https://nodejs.dev/)
- [PostgreSQL](https://www.postgresql.org/)

### Setup

1. Install dependencies: `npm install`
2. Setup database: `npm run db:up`

### Run

- Local development: `npm run dev`
- Production build: `npm run build && npm start`
